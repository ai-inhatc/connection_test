# 인공지능 기초 프로젝트 (인공지능기초)

이 프로젝트는 자동화된 빌드 및 테스트 워크플로우를 연습하기 위한 가상 프로젝트입니다.

## 프로젝트 구조

- **data/**: 데이터 파일을 저장하는 폴더입니다.
- **src/**: 프로젝트의 소스 코드가 포함되어 있습니다.
  - `math.js`: 산술 연산 로직을 담고 있습니다.
  - `app.js`: 애플리케이션의 메인 로직을 담당합니다.
- **tests/**: 단위 테스트 파일이 포함되어 있습니다.
  - `math.test.js`: `math.js`의 기능을 검증하는 테스트 파일입니다.
- **package.json**: 프로젝트 설정 및 실행 스크립트가 정의되어 있습니다.

## 주요 기능

1. **산술 연산**: `math.js`에서 제공하는 다양한 수학 함수.
2. **구문 체크**: 소스 파일의 문법적 오류를 사전에 점검합니다.
3. **단위 테스트**: 작성된 로직이 정상적으로 동작하는지 확인합니다.

## 실행 방법

### 빌드 (구문 체크)
```bash
npm run build
```

### 테스트 실행
```bash
npm run test
```
