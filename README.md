# 인공지능 기초 프로젝트 (인공지능기초)

이 프로젝트는 자동화된 빌드 및 테스트 워크플로우를 연습하기 위한 가상 프로젝트입니다.

## 최근 업데이트 내역
- **사용자 데이터 분석**: `data/users.json`에서 현재 활성 사용자를 필터링하여 출력하는 기능을 추가하였습니다.
- **수학 연산 유틸리티 확장**: 덧셈, 뺄셈 외에 곱셉(`multiply`)과 나눗셈(`divide`)을 추가하였습니다.
- **테스트 케이스 보강**: 새로운 연산 및 에러 핸들링(0으로 나누기) 테스트 케이스를 추가하였습니다.

## 프로젝트 구조

- **data/**: 데이터 파일을 저장하는 폴더입니다.
  - `users.json`: 샘플 사용자 정보를 담고 있습니다.
- **src/**: 프로젝트의 소스 코드가 포함되어 있습니다.
  - `math.js`: 산술 연산 로직을 담고 있습니다. (+, -, *, /)
  - `app.js`: 애플리케이션의 메인 로직을 담당하며, 활성 사용자 출력 기능을 포함합니다.
- **tests/**: 단위 테스트 파일이 포함되어 있습니다.
  - `math.test.js`: `math.js`의 모든 기능을 검증하는 테스트 파일입니다.
- **package.json**: 프로젝트 설정 및 실행 스크립트가 정의되어 있습니다.

## 주요 기능

1. **기초 산술 연산 유틸리티**: `math.js`에서 사칙연산 함수를 제공합니다.
2. **구문 체크**: 소스 파일의 문법적 오류를 점검합니다. (`npm run build`)
3. **사용자 필터링**: `users.json`의 상태(`status`) 값을 바탕으로 관리자를 위한 정보 가시화.
4. **단위 테스트**: 견고한 개발을 위한 종합 테스트 세트 운영.

## 실행 방법

### 애플리케이션 실행
```bash
node src/app.js
```

### 빌드 (구문 체크)
```bash
npm run build
```

### 테스트 실행
```bash
npm run test
```
